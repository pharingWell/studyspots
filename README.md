# For Developers

## Installation
___
1. `pip install -r requirements.txt`
2. `python3 manage.py migrate`
3. Set environmental vars DJANGO_SECRET_KEY, GOOGLE_API_KEY, GOOGLE_CLIENT_ID, and GOOGLE_SECRET to proper values
4. `python3 manage.py runserver`
